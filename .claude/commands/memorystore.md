# Memory Store Command

Update the project's CLAUDE.md and CLAUDELONGTERM.md files with information from this session.

## Instructions

1. **Review the current session** - Look at what was discussed, decided, and accomplished

2. **Read the current project memory files:**
   - `CLAUDE.md` in the project root (short-term, project-specific)
   - `CLAUDELONGTERM.md` in the project root (long-term patterns, architecture decisions)

3. **Update CLAUDE.md with:**
   - **Recent Changes** - What was modified this session
   - **Current Status** - Where the project stands now
   - **Next Steps** - What needs to be done next
   - **Session Notes** - Add a dated entry with key decisions/changes

4. **Update CLAUDELONGTERM.md with:**
   - **Architecture Decisions** - Any structural choices made
   - **Patterns & Conventions** - Code patterns established
   - **Known Issues** - Bugs or limitations discovered
   - **Feature Plans** - Long-term feature ideas discussed

5. **Keep it concise** - Use bullet points, no fluff
6. **Preserve existing content** - Append new entries, don't overwrite
7. **Show the user** - Display what was added/changed

After updating, confirm the changes were saved.
